总结平时实现业务的时候一些易错的点

  • es6默认值失效

    const user = {
  name: '',
  age: false,
  gender: 0,
  height: null,
  width: undefined
}
const {
  name = 'default',
  age = 'default',
  gender = 'default',
  height = 'default',
  width = 'default'
} =  user
console.log(name, age, gender, height, width) // '', false, 0, null, 'default'
// 原因是：转换成es5的代码 void 0 === 原值 ? 默认值 : 原值, 当且仅当原值为undefined才会取默认值
